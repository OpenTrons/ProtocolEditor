JSON Protocol Editor v0.0.1
Authored by Gordon Hart
June 2015


=========================
WHAT IT CAN DO:
=========================
- load and graphically display properly formatted JSON protocol files
- add containers, ingredients, and head items
- edit the names, values, and locations of deck, head, ingredient, and instruction items
- save changes as a download to the user's computer
- 


=========================
WHAT IT CANNOT DO:
=========================
- add new moves (transfer, distribute, consolidate, mix)
- add new attributes, locations to existing moves
- verify inputs: do typechecking on ints, booleans, well locations, etc.
- smart editing: bind all instances of a variable together and change each when one is edited
-
